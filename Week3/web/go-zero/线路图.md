# go-zero 开发框架学习路线图

![go-zero架构图](https://raw.githubusercontent.com/zeromicro/zero-doc/main/doc/images/architecture.png)
**学习路线图**
1. 基础篇（1-3天）
    - API语法与路由配置
    - RPC服务开发基础
    - 配置管理（yaml/json）
    - 日志系统配置

2. 进阶篇（4-15天）
    - 中间件开发机制
    - 分布式追踪整合
    - 服务熔断与限流
    - 数据库访问层生成
